# Lunch-Bot
A simple lunch bot
## Key Features
 - Gives information on the available lunch choices
 - User keys in the prefered lunch choice
 - If the choice selected is not available, a user is allowed to order again
 
 ## Learning Outcomes
 - Functions
 - Modules
 - while loops
 - Conditional statements.
